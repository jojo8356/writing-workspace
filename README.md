## Workflow recommandé

1. **Lire** `0_fondamentaux/pitch.md` → définir le projet en 1 phrase
2. **Remplir** `0_fondamentaux/cible.md` et `valeurs.md` → identifier le public et le message
3. **Construire** `1_personnages/protagoniste.md` → faire le test de robustesse
4. **Construire** `1_personnages/antagoniste.md` → faire le test de robustesse
5. **Définir** les règles de l'univers dans `2_univers/regles.md`
6. **Bâtir** la trame dans `3_trame/trame_globale.md`
7. **Cartographier** le temps dans `3_trame/timeline.md`
8. **Identifier** la scène centrale dans `0_fondamentaux/scene_centrale.md`
9. **Découper** en chapitres dans `4_plan/`
10. **Écrire** dans `5_ecriture/brouillon/`
11. **Réviser** avec les checklists de `6_revision/`
