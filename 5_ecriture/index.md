# 5. Écriture

> Le manuscrit en cours.

## Sous-dossiers

| Dossier | Rôle |
|---------|------|
| [`brouillon/`](brouillon/) | Le texte du roman, chapitre par chapitre |
| [`notes/`](notes/) | Idées brutes + déblocages quand bloqué |

## Convention pour les chapitres

- `brouillon/chapitre_01.md`
- `brouillon/chapitre_02.md`
- ...

## Template d'un chapitre

```markdown
# Chapitre __ — [Titre du chapitre si applicable]

> Référence au plan : 4_plan/plan_chapitre___.md
> Date de rédaction : ...
> Mots ciblés : ~3000-4000 (à ajuster selon ton format)

[Texte du chapitre]
```

## Discipline d'écriture

- 200 mots minimum par jour (cf. lacune identifiée dans Section I)
- Note dans `7_outils/suivi_quotidien.md` après chaque session
- Ne reviens PAS en arrière pendant le 1er jet (la révision arrivera en phase 6)
