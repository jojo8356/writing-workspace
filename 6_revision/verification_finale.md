# Vérification finale — Après le 1er jet complet

> **Section XIV du guide.** Les 12 points qui doivent tous passer avant de passer aux bêta-lecteurs.

## Les 12 points obligatoires

- [ ] **1.** Image d'ouverture et image de fin font écho
- [ ] **2.** Faiblesse fondatrice du héros attaquée à plusieurs reprises
- [ ] **3.** Tous les piliers narratifs plantés sont payés
- [ ] **4.** Le contrat émotionnel avec le lecteur est honoré (principe 31)
- [ ] **5.** Chaque perso important produit un changement (principe 4)
- [ ] **6.** Le pouvoir du héros a un coût visible et qui paie (principes 22, 27)
- [ ] **7.** Pas de "c'était un rêve" ni de retcon (principes 17, 25)
- [ ] **8.** Pas de jungle (le lecteur sait toujours où on va, principe 15)
- [ ] **9.** Cast resserré (principe 37) — tu n'as pas explosé le nombre de persos
- [ ] **10.** Les morts ont leur piste de décollage (principe 8)
- [ ] **11.** Le héros fait ses propres choix moraux critiques (principe 21)
- [ ] **12.** Les tons sont compartimentés, pas mélangés (principe 40)

## Tests supplémentaires

- [ ] J'ai relu en mode lecteur-architecte (cf. Section XI), pas seulement lecteur-acteur
- [ ] J'ai cherché spécifiquement les violations subtil + indirect (mon angle mort cognitif)
- [ ] Le rythme respire (silences entre intenses)
- [ ] Aucune scène ne peut être coupée sans que l'histoire s'appauvrisse

## Test final brutal

- [ ] Si je relis le roman dans 6 mois, est-ce que j'en serai fier ?
- [ ] Si quelqu'un que je respecte le lit, qu'est-ce qu'il en retiendra ?
