# Checklist — Fin de l'Acte II

## Vérifications de structure

- [ ] Premières conséquences du choix de l'Acte I montrées
- [ ] Nouveaux alliés/ennemis introduits
- [ ] Découvertes sur le monde / mystères secondaires
- [ ] Relation centrale approfondie
- [ ] Point milieu (50%) inverse une dynamique majeure
- [ ] Le héros change de façon visible (par les actes, pas par dialogue)
- [ ] Pression croissante
- [ ] Fausse victoire / fausse perte (~70%)
- [ ] "Tout est perdu" (75-80%) attaque la faiblesse fondatrice

## Principes critiques pour l'Acte II

- [ ] **Principe 35** — Le silence est utilisé pour le ménage psychologique (scènes calmes entre intenses)
- [ ] **Principe 7** — L'évolution du héros respecte le thème (hiérarchies bougent)
- [ ] **Principe 11** — La relation centrale est mise en perspective par d'autres relations
- [ ] **Principe 38** — Aucune transformation temporaire qui casse l'identification
- [ ] **Principe 40** — Compartimentation tonale (drame ≠ comédie dans la même scène)

## Test du rythme

- [ ] Y a-t-il des scènes calmes entre les scènes intenses ?
- [ ] Le rythme n'est pas un rollercoaster ininterrompu
- [ ] Le rythme n'est pas non plus une plaine sans relief
