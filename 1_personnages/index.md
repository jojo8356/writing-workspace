## Workflow de remplissage

1. Commence par `README.md` → liste tous les persos avec leur niveau
2. Remplis `protagoniste.md` → applique le test de robustesse (7 questions, section VI.C)
3. Remplis `antagoniste.md` → applique le test (7 questions, section VI.D)
4. Remplis `mentor.md` (s'il existe) → test du pilier (7 questions, section VI.E)
5. Remplis le reste

## Règles à respecter (rappels du guide)

- **Niveau 1** : 60-70% du livre, backstory maximale, transformation complète (principe 9, section X)
- **Niveau 2** : 20-30%, backstory substantielle, changement détectable
- **Niveau 3** : <10% chacun, une scène fondatrice suffit
- **Test final** : si tu enlèves un perso, qu'est-ce qui change dans le roman ? (principe 4)
