## Workflow type d'une session d'écriture

1. **Ouvrir** `7_outils/suivi_quotidien.md` → noter la date et l'objectif du jour
2. **Consulter** le plan du chapitre dans `4_plan/`
3. **Écrire** dans `5_ecriture/brouillon/`
4. **Mettre à jour** `7_outils/ponderation_vivante.md` si tu as ajouté/coupé des persos
5. **Noter** les idées qui surgissent dans `5_ecriture/notes/idees_brutes.md`
6. **Fermer** la session avec un mot dans `7_outils/suivi_quotidien.md`
