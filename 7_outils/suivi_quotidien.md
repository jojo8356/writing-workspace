# Suivi quotidien d'écriture

> Une entrée par session. Brève. Permet de garder le rythme.

## Discipline

- Objectif minimum : 200 mots / jour (cf. lacune Section I)
- Mieux : 500-1000 mots / session pour avancer un chapitre
- Hyper-bien : 2000+ mots / jour si tu peux maintenir

## Format d'une entrée

```markdown
### [Date]
- **Objectif visé** : ...
- **Mots écrits** : ...
- **Chapitre travaillé** : ...
- **Sentiment de la session** : (fluide / difficile / bloqué / inspirée)
- **Note pour la prochaine session** : ...
```
