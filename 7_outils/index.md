# 7. Outils — Référence quotidienne

## Fichiers

| Fichier | Quand l'utiliser |
|---------|-------------------|
| [`ponderation_vivante.md`](ponderation_vivante.md) | Mise à jour après chaque chapitre écrit |
| [`suivi_quotidien.md`](suivi_quotidien.md) | À la fin de chaque session d'écriture |
| [`ref_principes.md`](ref_principes.md) | Pendant l'écriture, pour vérifier un principe |
| [`ref_pieges.md`](ref_pieges.md) | Pour reconnaître les pièges classiques |

## Workflow quotidien

1. **Avant d'écrire** : ouvrir `suivi_quotidien.md`, noter date + objectif
2. **Pendant l'écriture** : consulter `ref_principes.md` si doute
3. **Après l'écriture** : mettre à jour `ponderation_vivante.md`, fermer `suivi_quotidien.md`
